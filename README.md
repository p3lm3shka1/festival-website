# Project for "Vilnius Coding School" #
## Festivalis TruLiaLia ##  


This project is a responsive website design made with nesting CSS. It includes different website sections like navigation, hero banner, gallery, tickets, contact form, and footer.
The design automatically changes to fit phones, tablets, and desktop screens.

## How It Works ##
<table>
  <ul>
<li>The website uses Flexbox for layout control.</li>
<li>dvw (dynamic viewport width) and svh (small viewport height) units make it flexible.</li>
<li>Hover effects use transitions for smooth animation.</li>
<li>The media queries ensure the design adapts between 375px and 768px.</li>
</ul>
</table>
